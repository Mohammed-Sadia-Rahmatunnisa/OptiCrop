🌱 About OptiCrop

OptiCrop is an AI-powered Smart Agricultural Production Optimization System that recommends the most suitable crop based on soil nutrients and environmental conditions.

The application uses Machine Learning algorithms including Random Forest, Decision Tree, Logistic Regression, KNN and K-Means to analyze agricultural data.

It helps farmers improve productivity, reduce crop failure, optimize fertilizer usage, and support sustainable agriculture.

✨ Features

🌱

AI Prediction

Predicts the best crop using Random Forest Machine Learning.

Farmer Friendly

Simple user interface with instant crop recommendations and farming guidance.
